Pipes: Allow you to transform the data before display

Hello, my name is {{myName}}

Hello, my name is {{ myName | lowercase }}

Hello, my name is {{ myName | uppercase }}

Hello, my name is {{ myName | titlecase }}

Hello, my name is {{ myName | slice:3:7 }}


Playing with Numbers

{{ 5.678 | number:'1.2-3'}}

{{ 5.678 | number:'3.1-2'}}

Percentage

{{ 5.678 | percent}}

Currency

{{ 123 | currency}}

{{ 123.456 | currency}}

{{ 123 | currency:'GBP'}}

Date

Date in full format : {{myDate}}

Date in short format: {{myDate | date:'shortDate'}}

Time in short format: {{myDate | date:'shortTime'}}

Some more examples of Pipe

using slice: My first name is {{myName | slice: 0:8 }}

Using json to display the value of the element of an object

{{ student }}

{{ student | json }}

Using number to display a big number in proper format using commas after every 3 digits from the right

{{ bigNumber | number }}
